# STANDARD-Gemini 3 Model Mandate

## 1. Exclusive Model Usage
This project **EXCLUSIVELY** uses the "Gemini 3" branding for all AI integration. 

- **FORBIDDEN**: "Gemini 2", "Gemini 2.5", "Gemini Flash 1.5" (in UI labels).
- **REQUIRED**: "Gemini 3 Flash Preview" and "Gemini 3 Pro Preview".

## 2. API Mapping (Provisional)
Until official "3.0" API endpoints are released, the following internal mappings are **MANDATED**:

- **Gemini 3 Flash Preview** -> `gemini-2.0-flash-exp` (or latest experimental high-speed model).
- **Gemini 3 Pro Preview** -> `gemini-1.5-pro-latest` (or latest available reasoning model).

## 3. UI Enforcement
All Dropdowns, Modals, and Config screens MUST use the "Gemini 3" nomenclature.
