# Visual Design Skill

Use this skill to generate visual assets (images, icons, textures) or design usage patterns.

## 🎨 Capabilities

### 1. Generate Assets (`generate_image`)

- **Favicons/Logos**: "Generate a pixel art favicon for a dungeon map."
- **Textures**: "Create a seamless stone texture for the Voxel Engine."
- **UI Mockups**: "Visualize a dark-mode login screen."

### 2. Design Philosophy

- **Aesthetic**: "Vibe Coding". Modern, clean, "Epic" gaming aesthetic.
- **Tooling**: Use `generate_image` tool directly.

## 📝 Best Practices

- **Prompting**: Be descriptive. Include style, mood, lighting, and resolution constraints.
- **Storage**: Save generated assets to `public/` or `src/assets/` immediately.
